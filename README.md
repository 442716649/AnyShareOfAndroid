# AnyShareOfAndroid
file transfer on android 仿茄子快传 或者 360文件传输 在局域网内，如果没有接收方建立热点，发送发接入热点，进行文件（发送方的app、图片等文件）的传输

## 注意
目前还在编码中，还是有bug的，因此只是将效果图放上来了。  我会对此项目持续进行更细，请关注。

# 效果图
需要两部手机，连接在同一个wifi环境下，会相互发现，并且通过点击发现后的对方，建立连接，进而进行文件的传输。
