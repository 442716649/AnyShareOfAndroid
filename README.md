# AnyShareOfAndroid
file transfer on android 仿茄子快传 或者 360文件传输 在局域网内，如果没有接收方建立热点，发送发接入热点，进行文件（发送方的app、音视频、图片等文件）

# 项目介绍
在局域网内，如果没有接收方建立热点，发送发接入热点，进行文件（发送方的app、音视频、图片）的传输。
